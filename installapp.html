Add to home screen to View the iPad Web App

This site was made as an iPad web app. If you need help installing this app follow the directions below.

  1. Click the icon with an arrow in a box. This should be to the left of the url.
  2. Choose add to Home screen.
  3. (optional) Change the name to anything you want it to say.
  4. Now click add.
  5. Safari will close and add an icon to your home screen.
  6. Click the icon to view the Web app made just for iPad.
